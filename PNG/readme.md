# PNG 
A PNG files is an image. The word PNG represents the acronym for *Portable Network Graphics*. The extension was created in March 1997 and is still a common file to stuble when using your device. PNG files like this can be used in various languages such as HTML, CSS and other similar web languages. The extension for PNG files is `.png`, preferably without capital letters.
